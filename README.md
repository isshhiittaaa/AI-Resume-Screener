# 📄 AI Resume Screening System

## 🚀 Overview
This project is an AI-based resume screening system that helps shortlist candidates by comparing resumes with a job description.

## 🔧 Features
- Resume scoring using TF-IDF
- Candidate ranking
- Strengths & gaps analysis
- Streamlit interactive UI
- Supports PDF & TXT resumes

## 🛠️ Tech Stack
- Python
- pandas
- scikit-learn
- Streamlit
- pdfplumber

## ▶️ How to Run
```bash
streamlit run app.py
